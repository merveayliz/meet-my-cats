# Meet My Cats: Murphy & Zuko 🐾✨

This is a personalized, responsive web project designed to showcase my two twin cats. It features a modern, mobile-first design using **HTML5** and **CSS3**, focusing on clean UI/UX principles and flexible layouts.

## 🚀 Live Demo
You can see the project live here: [https://merveayliz.github.io/meet-my-cats/](https://merveayliz.github.io/meet-my-cats/) 
*(Not: Kendi linkinle güncellemeyi unutma kanka!)*

## 🛠️ Features
- **Horizontal Card Layout:** Professional desktop view with images on the left and content on the right.
- **Responsive Design:** Optimized for mobile screens with a compact and sleek layout.
- **Glassmorphism:** Modern UI touches in the header and intro section.
- **Custom Branding:** Featured footer with "Designed by aylizdev" and social links.

## 📸 Project Glimpse
> Murphy is named after *The 100* and *Murphy's Law*. Zuko is named after *The Last Airbender* because of the orange spot on his ear! 🐈‍⬛🐈

## 💻 Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Flexbox, Grid, Media Queries, and custom variables.
- **Google Fonts:** Poppins for a modern look.
